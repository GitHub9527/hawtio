## Hawtio 2.x prototype

### To get going:

(use sudo for any global install as needed)

* `npm install -g gulp`

* `npm install -g bower`

* `npm install`

* `bower install`

### run the app:

* `gulp`


### Add a js dependency:

* `bower install --save my-awesome-dep`

* `gulp bower`

* commit the changed index.html, bower.json and karma.conf.js
